# Samsung

 Description :

An interactive Supply Chain & Logistics Analytics Dashboard developed using Microsoft Power BI to monitor revenue, inventory, shipment performance, supplier efficiency, and delivery insights. The dashboard transforms raw business data into actionable insights through dynamic visualizations and KPI tracking.

Tech Stack :

Excel Data Modeling
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Visualization

Data Source :
Kaggle Dataset

Features / Highlights : 

Interactive KPI Dashboard for Revenue, Profit, Inventory & Shipment Tracking
Supplier Performance & Lead Time Analysis
Inventory Monitoring & Product-Level Insights
Shipment Delay Analysis with Logistics Tracking
Customer Delivery Quantity & Revenue Analysis
Clean Dark-Themed Business Dashboard Design
Dynamic Charts & Data Visualization using DAX Measures
Data Cleaning & Transformation using Power Query
Business-Oriented Dashboard Layout with Supply Chain Flow Visualization

Key Metrics Covered: 

Gross Revenue
Total Revenue
Profit Margin %
Shipment Quantity
Inventory Value
Delivery Quantity
Net Supplier Revenue
Total Shipment Tracking
